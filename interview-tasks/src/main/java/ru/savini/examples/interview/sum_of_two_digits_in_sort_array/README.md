# Сумма двух чисел в массиве наиболее близкая к заданному числу

Условие             | Описание
------------------- | --------------------------------
Сложность алгоритма | Не сложнее O(n)

В сортированном массиве целых чисел требуется найти такую пару чисел,
сумма которой будет наиболее близка к заданному значению.

## Формат ввода
На вход подается сортированный целочисленный массив и заданное значение.

## Формат вывода
На выходе распечатать индексы пары чисел, сумма которых будет наиболее 
близка к заданному значению.

## Примеры
Ввод                                     | Вывод
---------------------------------------- | --------
`[-10, -5, 0, 1, 7, 15]`<br>`target = 0` | `{2, 3}`
`[-10, -8, -7, -5, 11]` <br>`target = 0` | `{0, 4}`
